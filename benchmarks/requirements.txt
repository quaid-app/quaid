# GigaBrain Advisory Benchmark Requirements
#
# Python dependencies for the advisory benchmark harnesses.
# These benchmarks run manually before major releases — they require API keys
# and/or a local Ollama instance. They are NOT required for CI.
#
# Install:
#   pip install -r benchmarks/requirements.txt
#
# Recommended: use a virtual environment
#   python -m venv .venv && source .venv/bin/activate
#   pip install -r benchmarks/requirements.txt
