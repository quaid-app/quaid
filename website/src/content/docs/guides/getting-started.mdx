## Choose a release channel

<div class="gb-guide-grid gb-guide-grid--two">
  <GuideFeatureCard icon={"download"} title={"Airgapped channel"}>
    <p>
      The default build. BGE-small weights are embedded into the binary so the
      whole stack works without any runtime network access.
    </p>
  </GuideFeatureCard>
  <GuideFeatureCard icon={"random"} title={"Online channel"}>
    <p>
      Smaller binary, same local behavior after first use. The model downloads
      once, caches locally, and stays private thereafter.
    </p>
  </GuideFeatureCard>
</div>

<CodePanel
  tone={'terminal'}
  label={'Source build'}
  code={`git clone https://github.com/macro88/gigabrain
cd gigabrain

# Airgapped channel

cargo build --release

# Online channel

cargo build --release --no-default-features --features bundled,online-model`}
/>

## Build your first brain

<div class="gb-guide-stack">
  <GuideStep
    number={'01'}
    title={'Initialize the database'}
    code={`gbrain init ~/brain.db`}
    codeLabel={'brain.db'}
    codeTone={'terminal'}
  >
    <p>This creates the full v4 schema: pages, embeddings, assertions, typed links, and the knowledge-gaps table.</p>
  </GuideStep>

<GuideStep
  number={"02"}
  title={"Import existing notes"}
  code={`gbrain import /path/to/notes/ --db ~/brain.db`}
  codeLabel={"Idempotent ingest"}
>
  <p>
    Import existing markdown safely. GigaBrain parses frontmatter, splits
    compiled truth from timeline, generates embeddings, and refreshes the
    database in place.
  </p>
</GuideStep>

  <GuideStep
    number={'03'}
    title={'Query, graph, and inspect'}
    code={`gbrain query 'who has worked with Jensen Huang?'
gbrain graph people/pedro-franceschi --depth 2
gbrain stats`}
    codeLabel={'Operate the brain'}
  >
    <p>Once content is loaded, you can move between semantic queries, graph exploration, and operational health checks without leaving the CLI.</p>
  </GuideStep>
</div>

## Connect an AI agent

<p class="gb-guide-section-intro">
  Any MCP-compatible client can connect over stdio. Start the server locally,
  point your client at <code>gbrain serve</code>, and keep the database path in{" "}
  <code>GBRAIN_DB</code>.
</p>

<CodePanel
  tone={"terminal"}
  label={"Serve locally"}
  code={`GBRAIN_DB=~/brain.db gbrain serve`}
/>
