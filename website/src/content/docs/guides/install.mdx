## Build from source

<p class="gb-guide-section-intro">
  If you want every feature immediately, build from source. SQLite and
  sqlite-vec are bundled, and the default binary embeds BGE-small weights for
  fully offline use.
</p>

<CodePanel
  tone={'terminal'}
  label={'Cargo'}
  code={`git clone https://github.com/macro88/gigabrain
cd gigabrain

# Airgapped channel — default

cargo build --release

# Online channel — downloads and caches BGE-small on first semantic use

cargo build --release --no-default-features --features bundled,online-model`}
/>

### Download a specific release

<CodePanel
  label={"GitHub Releases"}
  code={`VERSION='v0.9.1'
PLATFORM='linux-x86_64'   # linux-x86_64 | linux-aarch64 | darwin-arm64 | darwin-x86_64
ASSET='gbrain-\${PLATFORM}-airgapped'   # or: gbrain-\${PLATFORM}-online
curl -fsSL 'https://github.com/macro88/gigabrain/releases/download/\${VERSION}/\${ASSET}' -o '\${ASSET}'
curl -fsSL 'https://github.com/macro88/gigabrain/releases/download/\${VERSION}/\${ASSET}.sha256' -o '\${ASSET}.sha256'
shasum -a 256 --check '\${ASSET}.sha256'`}
/>

## Shell installer

<p class="gb-guide-section-intro">
  For most people, the shell installer is the cleanest path. It detects your
  platform, verifies the SHA-256 checksum, runs <code>gbrain version</code>, and
  prints the <code>GBRAIN_DB</code> shell-profile tip.
</p>

<CodePanel
  tone={'terminal'}
  label={'Installer'}
  code={`curl -fsSL https://raw.githubusercontent.com/macro88/gigabrain/main/scripts/install.sh | sh

GBRAIN_VERSION=v0.9.1 GBRAIN_CHANNEL=online \
 curl -fsSL https://raw.githubusercontent.com/macro88/gigabrain/main/scripts/install.sh | sh`}
/>
