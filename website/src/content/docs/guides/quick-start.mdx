## Fast path

<div class="gb-guide-stack">
  <GuideStep
    number={'01'}
    title={'Initialize a brain'}
    code={`gbrain init ~/brain.db`}
    codeLabel={'CLI'}
    codeTone={'terminal'}
  >
    <p>Create the SQLite-backed brain file that stores pages, links, assertions, and embeddings in one portable artifact.</p>
  </GuideStep>

  <GuideStep
    number={'02'}
    title={'Write a page'}
    code={`cat <<'MD' | gbrain put people/alice
---
title: Alice Example
type: person
---

# Alice Example

Above the line: compiled truth.

---

## Timeline

- 2026-04-14 — Met at a demo day; interested in offline-first knowledge systems.
  MD`}
  codeLabel={'Compiled truth + timeline'}

  >

      <p>Every page follows the compiled-truth model: the top stays current, the timeline remains append-only, and the whole thing is queryable immediately.</p>

    </GuideStep>

    <GuideStep
    number={'03'}
    title={'Search what you just wrote'}
    code={`gbrain search 'offline-first'
gbrain query 'who is interested in offline knowledge systems?'`}
    codeLabel={'Keyword + semantic'}
  >
    <p>Use <code>search</code> for FTS5 keyword retrieval and <code>query</code> for hybrid semantic retrieval with Reciprocal Rank Fusion.</p>
  </GuideStep>
</div>

## Install options

<div class="gb-guide-grid gb-guide-grid--two">
  <GuideFeatureCard icon={"download"} title={"Build from source"}>
    <p>
      Available now. The default build is airgapped and embeds the BGE-small
      model weights directly into the binary.
    </p>
  </GuideFeatureCard>
  <GuideFeatureCard icon={"cloud-download"} title={"GitHub Release binaries"}>
    <p>
      Available now for macOS ARM/x86 and Linux x86_64/ARM64 in both airgapped
      and online channels.
    </p>
  </GuideFeatureCard>
  <GuideFeatureCard icon={"rocket"} title={"Shell installer"}>
    <p>
      Available now. The installer defaults to the airgapped channel and
      switches to the online build with <code>GBRAIN_CHANNEL=online</code>.
    </p>
  </GuideFeatureCard>
  <GuideFeatureCard icon={"warning"} title={"npm global install"}>
    <p>
      Implemented and staged. It will default to the online channel when the
      public package rollout opens.
    </p>
  </GuideFeatureCard>
</div>
