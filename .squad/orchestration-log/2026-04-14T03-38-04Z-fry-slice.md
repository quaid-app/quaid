3. **`Link` uses slugs, not page IDs** — DB layer resolves to IDs internally
4. **`i64` for all DB integers** — matches SQLite INTEGER type
5. **Module-level `#![allow(dead_code)]`** — temporary; remove when db.rs wires types
6. **`SearchMergeStrategy::from_config`** — parses config table strings with SetUnion default
