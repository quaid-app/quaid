## The information sovereign

<p class="gb-guide-section-intro">
  The current landscape of knowledge management is fractured by dependency. We
  rent our memory from services that can change policy, disappear, or decide
  that semantic features now require yet another key and another bill.
</p>

<p>
  <strong>GigaBrain</strong> flips that arrangement. It is an architectural
  approach to software — built with the same philosophy as a physical library:
  heavy, stable, inspectable, and entirely yours.
</p>

<blockquote class="gb-guide-quote">
  Intelligence should not require a handshake with the cloud.
</blockquote>

<p>
  It is also built for an agent-first world. The same single-file brain can
  answer full-text searches, semantic queries, graph exploration, contradiction
  checks, and MCP tool calls without handing custody of your data to anyone
  else.
</p>

<p>
  And now it stays current. The v0.9.6 release adds a live file
  watcher that follows your Obsidian vault as you edit — reconcile-backed, with
  1.5 s debounce and write-safety interlocks — so the brain that an AI agent reads
  is always the same brain you just updated. <em>(Unix/macOS/Linux only)</em>
</p>
